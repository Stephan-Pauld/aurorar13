1. Work Performed – Diagnostic

Diagnostic Fee: $49.00 (applied toward repair if approved)

Findings

  • System arrived with symptom: powers on, lights and fans active, but no video output.
  • Tested with client's original RTX 3090 and a known-good GeForce RTX 2060 Super – no video in either case.
  • Tested system with all storage removed (M.2 NVMe and 2.5" SSD). System successfully POSTs, displays video, and enters Alienware SupportAssist / BIOS setup.
  • Reinstalling the original M.2 NVMe drive immediately causes a no‑POST, no‑video condition again, even with a known-good GPU installed.
  • CMOS/RTC reset performed; with storage removed the system is stable and repeatedly POSTs, confirming motherboard, CPU, RAM, PSU, and GPU are functional.

Diagnostic Conclusion: The primary fault is a failed M.2 NVMe SSD (system drive). When installed, this drive prevents the system from completing POST, which is why no video output was present.

2. Recommended Repair – Restore Functionality

The following work is recommended to return the system to normal operation:

  • Supply and install a new M.2 NVMe SSD (system drive).
  • Install Microsoft Windows 11 and perform basic system configuration.
  • Verify stable boot to Windows and confirm GPU output is working as expected.

4. Optional Service – Data Recovery Attempt

Your original M.2 NVMe system drive is failing in a way that prevents the computer from even starting when it is installed. This indicates a probable controller or firmware‑level failure on the SSD.

Important: In this failure mode, data recovery is not guaranteed. Basic software‑based recovery tools can only help if the drive can still be detected and accessed in some form by another system (for example, via an external NVMe enclosure). If the drive is electrically dead or not detected at all, recovery typically requires a specialized data recovery lab, which is a separate, higher‑cost service.

Option A – In‑House Best‑Effort Recovery

  • Test the failed NVMe in an external enclosure on a separate system.
  • If detectable, attempt to copy user data (Documents, Desktop, Pictures, etc.) to an external drive using specialized recovery tools.
  • Billable as best‑effort labour only; success is not guaranteed.
